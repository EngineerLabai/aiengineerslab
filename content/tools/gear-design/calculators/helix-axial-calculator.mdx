## Calculation Explanation

<Callout type="info" title="How to use?">
<StepList title="Quick steps" steps={["Enter power & speed or torque", "Set pitch diameter and angles", "Read axial and radial forces"]} />
</Callout>

<Formula label="Helical gear forces">
T = 9550 · P / n
Ft = 2000 · T / d
Fa = Ft · tan(β)
Fr = Ft · tan(α)
</Formula>

**Assumptions**
- Steady torque with constant geometry.
- Pitch diameter is used for force calculation.

**Units**
- P: kW, n: rpm, T: N·m
- d: mm
- Forces: N

**References**
- ISO 6336 (Cylindrical gears)
- AGMA 2001 (Fundamental gear design)

<Callout type="warning" title="Out of scope / Warning">
Does not include dynamic load factors or bearing reactions. Use full gear calculations for final sizing.
</Callout>

## Examples

### Example 1: Helical power case

**Inputs**
- P = 7.5 kW, n = 1450 rpm
- d = 140 mm, α = 20°, β = 15°

**Outputs**
- T = 49.40 N·m
- Ft = 705.7 N
- Fr = 256.8 N
- Fa = 189.1 N

### Example 2: Torque input

**Inputs**
- T = 80 N·m
- d = 100 mm, α = 20°, β = 30°

**Outputs**
- Ft = 1600 N
- Fr = 582.4 N
- Fa = 923.8 N
