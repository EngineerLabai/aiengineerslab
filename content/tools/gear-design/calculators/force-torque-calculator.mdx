## Calculation Explanation

<Callout type="info" title="How to use?">
<StepList title="Quick steps" steps={["Enter power & speed or direct torque", "Set pitch diameter and angles", "Read Ft, Fr, Fa results"]} />
</Callout>

<Formula label="Gear forces">
T = 9550 · P / n
Ft = 2000 · T / d
Fr = Ft · tan(α)
Fa = Ft · tan(β)
</Formula>

**Assumptions**
- Steady torque, no dynamic factors.
- Pitch diameter is used for force calculation.

**Units**
- P: kW, n: rpm, T: N·m
- d: mm
- Forces: N

**References**
- ISO 6336 (Cylindrical gears)
- AGMA 2001 (Fundamental gear design)

<Callout type="warning" title="Out of scope / Warning">
Dynamic factors, load distribution, and safety factors are not included. Use detailed gear standards for final design.
</Callout>

## Examples

### Example 1: Power input

**Inputs**
- P = 5 kW, n = 1500 rpm
- d = 120 mm, α = 20°, β = 0°

**Outputs**
- T = 31.83 N·m
- Ft = 530.6 N
- Fr = 193.1 N
- Fa = 0 N

### Example 2: Torque input

**Inputs**
- T = 120 N·m
- d = 150 mm, α = 20°, β = 15°

**Outputs**
- Ft = 1600 N
- Fr = 582.4 N
- Fa = 428.7 N
