# Ornekler

## Ornek 1: LMTD

LMTD hesaplama.

**Girdiler**
- Th_in 120 C
- Th_out 80 C
- Tc_in 20 C
- Tc_out 50 C

**Ciktilar**
- LMTD ~65 C

**Notlar**
- Cok gecisli ise duzelt.

## Ornek 2: Iletim

Iletimle isi transferi.

**Girdiler**
- k=45 W/mK
- Alan 0.2 m2
- Kalinlik 0.01 m
- DeltaT 40 C

**Ciktilar**
- Isi transferi ~36 kW

**Notlar**
- Isil temas direncini unutma.
